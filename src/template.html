My Template Website

Some notes to help me remember what this contains:

  • Built with npm / webpack
  • Includes package.json and webpack.config.js files
  • Source from Odin Project -- visit that page to get the key terminal commands to run it
  • Includes reset.css file from Josh Comeau
  • Webpack uses index.js as the entry point. That index.js file imports the CSS stylesheets and any other js files
  • Other js files can be downloaded from npm and included in that index.js file
  • "npm run dev" terminal command spins up a live web server on localhost:8080
